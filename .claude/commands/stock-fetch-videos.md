# 신규 증시 영상 감지

이 스킬은 설정된 유튜브 채널에서 신규 영상을 감지하고 키워드 필터링합니다.

## 수행 절차

### 1. RSS 피드에서 신규 영상 가져오기

```bash
python3 scripts/fetch_rss.py
```

이 스크립트는:
- `stock-config.json`에서 채널 목록과 키워드 설정을 읽습니다.
- 각 채널의 YouTube RSS 피드를 조회합니다.
- 채널별 키워드로 제목을 필터링합니다 (키워드가 비어있으면 전체 통과).
- `data/processed_videos.json`과 대조하여 이미 처리된 영상을 제외합니다.
- JSON 배열로 신규 영상 목록을 출력합니다.

### 2. 결과 보고

출력된 JSON 배열을 파싱하여 사용자에게 보기 좋게 정리합니다:

- 신규 영상이 있으면: 채널별로 그룹핑하여 제목, URL, 게시일을 테이블로 보여줍니다.
- 신규 영상이 없으면: "신규 영상이 없습니다." 라고 보고합니다.

## 출력 형식 예시

```
### 신규 영상 감지 결과

**한경글로벌마켓** (2건)
| 제목 | 게시일 |
|------|--------|
| [김현석의 월스트리트나우-4월9일] ... | 2026-04-08 |
| [빈난새의 개장전요것만-4월8일] ... | 2026-04-08 |

**증시각도기TV** (1건)
| 제목 | 게시일 |
|------|--------|
| [4월 9일 미국시황] ... | 2026-04-08 |

총 3건의 신규 영상이 발견되었습니다.
```
