# 증시 영상 처리 (자막 추출 + 요약)

이 스킬은 단일 유튜브 영상의 자막을 추출하고 증시 투자자 관점에서 요약합니다.

## 인자

- `$ARGUMENTS`: 처리할 YouTube URL (예: https://www.youtube.com/watch?v=VIDEO_ID)

## 수행 절차

### 1. 영상 타입 확인

```bash
mmk youtube videotype $ARGUMENTS
```

- Shorts 영상이면 "Shorts 영상은 스킵합니다." 라고 보고하고 종료합니다.

### 2. 메타데이터 조회

```bash
mmk youtube metadata $ARGUMENTS --output json
```

- 제목, 채널명, 조회수, 게시일 등을 파싱합니다.

### 3. 자막 추출

```bash
mmk youtube transcript $ARGUMENTS --format text
```

- 자막이 없는 경우 "자막을 추출할 수 없습니다." 라고 보고하고 종료합니다.
- 자막이 너무 길면 (50,000자 초과) 앞부분 50,000자만 사용합니다.

### 4. 요약 생성

추출한 자막과 메타데이터를 바탕으로 다음 구조의 한국어 요약을 생성합니다:

```
## 한줄 요약
(영상의 핵심 메시지를 1문장으로)

## 핵심 포인트
- (투자자에게 중요한 포인트 3~5개)

## 언급 종목/섹터
- (영상에서 언급된 종목명, 티커, 섹터 나열)

## 시장 전망
(긍정적 / 부정적 / 중립) — 근거 1~2문장

## 투자 시사점
- (실질적인 투자 인사이트 1~3개)
```

요약 시 유의사항:
- 한국어로 작성합니다.
- 투자자 관점에서 실질적으로 유용한 정보에 집중합니다.
- 종목명은 한국어와 영문 티커를 함께 표기합니다 (예: 삼성전자(005930), 테슬라(TSLA)).
- 주관적 추천이 아닌 객관적 정보 요약에 집중합니다.

### 5. 결과 출력

메타데이터와 요약을 합쳐서 출력합니다:

```
---
제목: [영상 제목]
채널: [채널명]
URL: [유튜브 링크]
게시일: [날짜]
조회수: [조회수]
---

[요약 내용]
```
